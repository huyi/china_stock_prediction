# 三、结构阶段判断

当前市场形成：

> 4 条主链 + 1 条弱链结构

### 主链结构：

1. 资源主线（稀缺资源 / 稀土永磁 / 小金属概念）
2. 电力升级链（抽水蓄能 / 超超临界发电 / 绿色电力）
3. 储能材料链（钒电池 / 锂矿概念）
4. 化工材料链（化工原料 / 煤化工 / 钛白粉）

### 弱链：

- 安全防务扩散（军工 / 民爆概念 / 中字头）

- 市场不是单一板块独立趋势，呈现“多链并行共振”
