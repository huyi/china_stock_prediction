# 三、结构阶段判断

当前市场形成：

> 4 条主链 + 1 条弱链结构

### 主链结构：

1. 新材料链（复合集流体为核心）
2. 电力升级链（超超临界发电 + 抽水蓄能）
3. 新能源设备链（风电 / 光伏辅材）
4. 能源服务链（工业气体 / 油气设服）

### 弱链：

- 电子扩散（PCB）

# 四、结构特征总结

- 市场不再由单一材料链主导  
- 多链并行共振，分散度提升  
- 资源链仅保留代表性暴露  
- 当前为“多主线强化阶段”，非单板块独立趋势
